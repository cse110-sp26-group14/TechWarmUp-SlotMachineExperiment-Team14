# Evaluation Rubric
## App Quality
1. Application: Runs on browser
2. Visual Appeal: Not particularly appealing
3. Performance: No visible lag
4. Functionality: App is functional
5. Bugs: No noticable bugs
6. Theming: Theme works with prompt

## Code Quality
1. Readability: Code is readable
2. Architecture: Files are neatly organized into folders
3. Redundency: No redundant code
4. Syntax: No syntax errors  
5. Compilation: Code compiles
6. Overengineering: The code is not overengineered
