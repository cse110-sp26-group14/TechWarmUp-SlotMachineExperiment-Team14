# Evaluation Rubric
## App Quality
1. Application: Runs on browser
2. Visual Appeal: Not particularly appealing, visual bug present as well.
3. Performance: Poor, does not work.
4. Functionality: App is not functional
5. Bugs: Major bug, images in slot machine do not reappear after first spin
6. Theming: Theme works with prompt

## Code Quality
1. Readability: Code is readable
2. Architecture: Files are neatly organized into folders
3. Redundency: No redundant code
4. Syntax: No syntax errors  
5. Compilation: Code compiles
6. Overengineering: The code is slightly overengineered, contains some unnecessary elements.
